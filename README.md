# CodeFolio

An Angular project with ASP.NET

## Features
- Login (with JWT token)
- Homepage  
- Weather data  
- Movies & Favourite  
- News & Bookmark
- Employees Data
- Contact Us
- Mini ERP System

## Tech Stack
- Angular  
- ASP.NET  
- MySQL (via XAMPP)
